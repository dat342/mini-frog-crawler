Mini Frog — Tool quét Web URL

Crawl theo thứ tự: Trang chủ → Danh mục/Dịch vụ → Sản phẩm → Blog · chạy local, không cần API

Mẹo: copy cột URL từ Excel/Google Sheets rồi dán thẳng vào đây. Giới hạn 5.000 URL.

Chức năng này CHỈ tái tạo URL dạng %category%/%post% và phân loại trang — không bóc SEO. Nguồn ưu tiên: Schema breadcrumb → Breadcrumb HTML → article:section → WordPress API.

⚙ Pattern phân loại trang (sửa nếu website đặt URL khác)

Mỗi dòng một đoạn URL. URL chứa đoạn nào sẽ được xếp vào loại đó. Ưu tiên: Danh mục → Blog → Sản phẩm.

URL khớp bất kỳ mẫu nào sẽ bị bỏ qua. Không áp dụng cho chế độ "Chỉ đúng URL tôi dán".
